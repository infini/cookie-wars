# 쿠키전쟁 (Cookie Wars)

아이가 기획한 손그림을 바탕으로 만든 Android 전용 세로형 싱글 플레이 게임입니다. 쿠키를 모으는 화면과 전투 화면을 완전히 분리했으며, 화면 아래의 게임·전투·강화·정보 4개 대분류와 간결한 소메뉴로 이동합니다.

## 설치

- APK: [`artifacts/cookie-wars-v1.0.15.apk`](artifacts/cookie-wars-v1.0.15.apk)
- APK 크기: 103,682,798바이트 (98.88MiB)
- SHA-256: `40b98260942e0f2bc9f3361a6875b106f901ae44f09fa0ef5a7fe119c9acbfe1`
- 패키지명: `com.cookiewars.game`
- 지원 범위: Android 7.0(API 24) 이상
- 화면 방향: 세로 고정

APK를 Android 스마트폰으로 옮긴 뒤 파일을 열어 설치합니다. Play 스토어가 아닌 APK를 직접 설치하므로, 스마트폰에서 해당 파일 관리 앱의 **알 수 없는 앱 설치 허용**이 필요할 수 있습니다.

현재 APK는 직접 테스트·설치를 위한 개발 서명 빌드입니다. Play 스토어 배포 전에는 별도의 안전한 배포 키로 서명해야 합니다.

## 현재 구현

- 가운데 전체 쿠키 클릭 영역, 클릭 애니메이션, 획득 텍스트, 효과음·진동
- 화면에 표시되는 데이터 기반 무한 강화 5종, 진화 기여 플래그와 기존 크기 레벨의 1회 저장 이전
- 별도 경험치 없이 진화 기여 강화 레벨과 기존 저장 보너스의 합계로 자동 진화하는 50종 쿠키
- 기본 확률 1%·획득량 10배의 쿠키 크리티컬과 확률 50% 상한·배수 무제한의 무한 강화
- 기본 확률 0.1%·획득량 100배, 강화당 확률 +0.025%p가 기존 레벨에도 소급되는 배타적 쿠키 슈퍼 크리티컬과 확률 10% 상한·배수 무제한 강화
- Freesound CC0 `Crunch`를 세 가지 속도로 번갈아 쓰는 쿠키 클릭음과 Mixkit 충격·반짝임을 겹친 크리티컬 사운드
- 원형 링 없이 교차 참격광·갈라지는 번개·각진 쿠키 파편을 합성하고 연속 발동은 축약하는 크리티컬 이펙트
- X자 다색 참격·수직 낙뢰·10갈래 번개·파편 폭발·쿠키 무대 흔들림과 Mixkit `콰광` 전기 폭발을 합성한 슈퍼 크리티컬 피드백
- 터치 시작 즉시 처리, 넓은 hit slop·이동 허용 범위와 동일 제스처 중복 방지로 개선한 쿠키 클릭 입력
- 메인 화면에서 크게 확인하는 현재 진화 레벨·다음 쿠키 필요 레벨·남은 강화 수
- 길 없이 넓은 중앙 공간을 유지하는 15개 난이도별 고유 전장
- 넓어진 공격 반경 안의 적만 노리는 쿠키봇·쿠키 성 원반
- 성 앞에서 멈춘 적도 원반 속도와 관계없이 반드시 명중하는 추적 충돌
- 느린 비행·회전으로 궤적이 보이는 원반과 작은 원근 유닛
- 파랑·금색 아군 원반과 빨강·검붉은색 적 원반
- 적 수는 항상 한 마리이며 보스 능력치로 조절되는 난이도
- 아군 이름 파란색, 적군 이름 빨간색 표시
- 저렴하게 구매 가능하고 등급별 능력 격차가 큰 쿠키 원반 5종과 쿠키봇 5종
- 강화 아래에서 쿠키·원반·쿠키봇 강화를 바로 고르는 3개 소메뉴
- 정보 아래에서 쿠키 도감·난이도·몬스터를 바로 고르는 3개 소메뉴와 NEW 배지
- 상위 종류로 갈수록 큰 폭으로 벌어지는 원반·쿠키봇 공격 능력
- 종류별 영구 소유·장착과 제한 없는 원반 강화
- 각 봇 위치에서 나오는 작은 자동 원반과 전투 화면 어디든 누르면 성에서 나오는 큰 2배 원반
- 성벽·탑·쿠키 문양을 갖춘 별도 쿠키 성 오브젝트
- 쿠키를 들고 싸우며 실제 위치를 순찰하고 원반을 던지는 5종 쿠키봇, 종류별 달리기 3프레임·투척 3프레임 애니메이션
- 15단계 난이도 드롭다운과 난이도별 20승 순차 해금
- 같은 난이도에서도 승리마다 강해지는 1~20번째 전투
- 난이도마다 실루엣·갑옷·망치가 다른 고유 보스 15종과 보스별 걷기 3프레임·망치 강공격 3프레임 애니메이션
- HP·공격력·이동 속도와 적 원반 레벨로 상승하는 보스 난이도
- 성장한 쿠키봇 군단의 실제 DPS를 반영하는 단일 보스 즉사 방지 HP 하한
- 기본 공격력·공격 속도 2배, 이동 속도 20% 감소와 최종 난이도 20% 상향을 적용한 원거리·근접 보스 공격
- 화면 상단의 긴 보스 HP 바, 공격 예고·돌진, HP 50% 분노, 여러 좌표의 연속 충격·피해 숫자·적 원반 궤적
- 전장 입장부터 각자 고유한 거대 전쟁망치를 든 보스와 공격 사거리 진입 후 5초 주기의 `special` 강공격에서만 재생되는 망치 준비·내려찍기·복귀 연출
- 저장되는 X1·X2·X3 전투 속도와 배수만큼 실제 전투 시뮬레이션을 진행하는 서브스텝
- 저장되는 자동 전투 ON/OFF, 성의 쿨타임별 자동 공격과 승리 보상 확정 뒤 다음 스테이지 연속 진행
- 단순 색 변형이 아닌 초원·과수원·설원·정글·폭풍 절벽·악마계·신계를 아우르는 15개 독립 배경 테마
- 각 전투 최초 클리어마다 받는 화면 1/3 크기의 최강 쿠키봇 원반 기준 30배 거대 원반 1개
- 새 스테이지를 진행해 이길 때마다 받는 영구 전투 훈장 1개와 훈장당 클릭 힘·자동 생산·쿠키 성 체력 각 +1%
- v9 이전 저장의 난이도별 완료 스테이지 수를 합산해 누락된 전투 훈장을 한 번 자동 소급 지급
- 사용 시 보유량이 1개 줄고 두 겹 오라와 발광 회전 효과가 나타나는 거대 원반
- CC0 오케스트라 전투 음악과 아군·적군 발사, 약·강 피격, 근접·분노를 구분한 다층 효과음
- 전투 종료 즉시 음악·액션음 정지, 중복 액션음 재생 제한
- 결과 화면의 로비 이동·승리 시 다음 전투·패배 시 다시 전투 바로가기
- 각 난이도의 20번째 전투를 처음 이기면 다음 난이도를 자동 선택하고 `다음 전투`로 새 난이도 1번째 전투 시작
- 외곽선과 남은 체력에 따라 빨강에서 초록으로 변하는 체력 게이지
- 구매 가능한 강화를 항상 위에, 완료 강화를 아래에 두는 실시간 정렬
- 각 난이도의 1~20 전투를 처음 클리어할 때마다 거대 원반·전투 훈장 보상, 완료 전투 재도전은 보상 없음
- 전투 승리로 쿠키를 직접 지급하지 않고, 전투 훈장이 쿠키 획득·생산·성 체력을 영구 강화
- 몬스터 도감의 `NEW` 및 잠금 표시
- 앱 종료 후 진행 상황 자동 저장과 저장 시각부터 재실행까지 누락 없이 정산되는 자동 생산
- 저장 스키마 v11과 전투 속도·자동 전투·두 크리티컬 강화 진행의 안전한 복구
- 설정에서 효과음 5단계 볼륨과 사운드·진동 개별 켜기/끄기
- 모든 사용자 화면을 한국어로 구성

## 개발과 빌드

Node.js와 Android SDK/JDK가 설치된 환경에서 실행합니다.

```bash
npm install
npm run verify
npm run prebuild
npm run android:release
```

빌드 결과는 `android/app/build/outputs/apk/release/app-release.apk`에 생성됩니다. 실제 기기에서 개발 모드로 실행하려면 USB 디버깅을 켜고 `npm run android:device`를 사용합니다.

## 구조

```text
src/
├── components/   공통 모바일 UI
├── config/       난이도·몬스터·업그레이드·무기 JSON 데이터
├── domain/       성장·진행·보상 순수 계산
├── engine/       프레임 기반 전투 로직
├── navigation/   4대 메뉴·소메뉴 데이터와 검증 모델
├── screens/      8개 leaf 화면과 전투 화면 전용 모듈
├── services/     저장, 효과음, 진동
├── state/        영구 상태·저장 이전·명령·자동 생산
├── theme/        색상과 글꼴
└── types/        전투·성장·저장·UI 영역별 TypeScript 데이터 계약
```

- [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md): 코드 계층, 상태 변경, 저장과 확장 구조
- [`docs/CODE_REVIEW.md`](docs/CODE_REVIEW.md): 1.0.13 코드 리뷰 결과, 해결 사항과 후속 점검
- [`docs/DATA_TABLES.md`](docs/DATA_TABLES.md): 모든 JSON 테이블의 필드와 수정 방법
- [`docs/ASSET_LICENSES.md`](docs/ASSET_LICENSES.md): 외부 리소스 출처와 라이선스
- [`docs/GENERATED_ASSETS.md`](docs/GENERATED_ASSETS.md): 전장·보스·쿠키봇·쿠키 생성 프롬프트, 변환 절차와 검수 기준
- [`docs/DEFENSE_GAME_RESEARCH.md`](docs/DEFENSE_GAME_RESEARCH.md): 디펜스 탑 10 비교, 타격감·공격 연출 적용 근거

## 주요 명령

- `npm start`: Expo 개발 서버
- `npm run android:device`: 연결된 Android 기기에서 실행
- `npm run verify`: TypeScript, Jest, Expo 프로젝트 진단
- `npm run android:release`: Android APK 생성
