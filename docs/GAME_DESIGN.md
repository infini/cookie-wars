# 쿠키전쟁 게임 상세 명세

## 제품 정보

- 이름: 쿠키전쟁 (Cookie Wars)
- Android 패키지: `com.cookiewars.game`
- 플랫폼: Android 전용 APK
- 화면: 스마트폰 세로 고정
- 플레이: 한국어 싱글 플레이
- 대상: 어린이. 큰 터치 영역, 짧은 문장, 아이콘 중심, 밝은 쿠키 색상을 사용

## 핵심 흐름

```text
게임 화면에서 쿠키 클릭
→ 쿠키 획득
→ 쿠키 업그레이드 또는 원반·쿠키봇 구매
→ 난이도 선택
→ 전투 화면에서 자동 전투
→ 승리 횟수 기록 및 최초 보상 판정
→ 메인 게임 화면 복귀
```

게임 화면에는 적, 원반, 쿠키봇, 전투가 나타나지 않습니다. 전투 화면에서만 적, 봇 자동 공격, 원반, 승패가 작동합니다.

## 공통 내비게이션

모든 일반 화면 하단에 게임, 전투, 쿠키, 쿠키 업그레이드, 몬스터, 난이도, 원반, 쿠키봇 메뉴를 같은 순서로 표시합니다. 원반과 쿠키봇은 서로 분리된 독립 화면이며, 중간 탭을 거치지 않고 한 번 눌러 이동합니다.

## 게임·쿠키 성장

가운데 큰 쿠키를 누르면 현재 `clickPower`만큼 쿠키가 증가하고 클릭 애니메이션, `+수량` 텍스트, 효과음, 약한 진동이 발생합니다. 자동 생산 업그레이드는 설정된 주기마다 쿠키를 더합니다.

쿠키 업그레이드는 크기, 클릭당 획득, 자동 생산, 쿠키 성 체력으로 구성합니다. 각 카드에는 현재 단계·현재 능력·다음 능력·가격을 표시합니다. 클릭 힘, 자동 생산, 쿠키 성 체력은 최고 레벨 없이 계속 강화되고 쿠키 크기만 명시된 마지막 단계에서 멈춥니다. 초기 값은 `cookie-upgrades.json`, 이후 무한 성장량은 `cookie-upgrade-rules.json`에서 수정합니다.

네 가지 업그레이드의 현재 레벨 합계가 진화 조건에 도달하면 쿠키가 자동으로 더 좋은 종류로 바뀝니다. 쿠키는 클래식 초코칩, 행운, 도넛, 와플, 컵케이크, 딸기 케이크, 달빛, 파티, 별빛 쌀쿠키, 로열 초콜릿의 10단계입니다. 높은 쿠키일수록 클릭 획득·자동 생산·쿠키 성 HP에 더 큰 배율을 줍니다. 쿠키 화면에서 현재 종류, 다음 조건, 남은 총레벨, 전체 진화 도감을 확인합니다.

## 전투

아군은 화면 아래, 적군은 위에서 시작합니다. 모든 아군 개체 이름은 파란색, 모든 적 개체 이름은 빨간색으로 객체 바로 위에 표시합니다. 적은 한꺼번에 몰려 나오지 않고 첫 적부터 일정 간격으로 한 마리씩 등장합니다. 등장 위치는 화면 위쪽의 균일한 가로 슬롯을 순환합니다. 적은 중앙 아래의 쿠키 성을 향해 이동하며 원반과 근접 공격으로 성의 체력을 줄입니다.

쿠키봇은 구매한 수량만큼 장착 원반의 피해가 합산되며 가장 가까운 살아 있는 적에게 쿠키 원반을 자동으로 던집니다. 전투 화면의 쿠키봇 그림은 쿠키를 들고 있습니다. 자동 공격은 항상 작동하며 끌 수 없습니다.

### 쿠키봇 자동 공격과 쿠키 성 공격

자동/수동 모드 선택은 없습니다. 쿠키봇은 종류별 공격 간격에 맞춰 장착 원반을 자동으로 발사합니다. 같은 종류의 봇을 여러 대 보유하면 한 발의 피해가 수량만큼 합산됩니다.

쿠키 성은 절대로 자동 공격하지 않습니다. 사용자가 전투 화면 아래의 쿠키 성 자체를 눌렀을 때만 장착 원반을 발사하며, 이 원반의 피해는 쿠키봇 원반의 2배입니다. 이전 원반이 화면에 남아 있어도 쿨타임만 끝나면 다음 원반을 발사할 수 있습니다.

전투 시작에는 장착할 원반과 쿠키봇이 각각 하나 이상 필요합니다. 쿠키 성은 성벽과 세 개의 탑이 보이는 별도 성 그래픽이며 중앙에 현재 진화 쿠키 문양을 표시합니다. 업그레이드와 현재 쿠키 배율로 증가하는 HP가 있고 적 원반과 근접 공격에 피해를 받습니다. 성 HP가 0이면 아군 패배, 모든 적 HP가 0이면 아군 승리입니다.

## 원반·쿠키봇 상점

원반 화면에는 원반 10종만, 쿠키봇 화면에는 쿠키봇 10종만 표시합니다. 모두 비교적 낮은 초기 가격으로 시작하며 봇의 다음 구매 가격은 테이블 배율로 완만하게 증가합니다. 원반은 종류별로 영구 소유하고 하나를 장착합니다.

원반은 상위 종류로 갈수록 Lv.1 공격력·크기·속도가 큰 폭으로 오르고 쿨타임은 짧아집니다. 쿠키봇도 종류별 원반 피해 배율과 자동 발사 간격 차이가 크며, 화면 카드에서 두 능력치를 함께 비교합니다. 봇 색상, 가격과 모든 능력치는 `bots.json`에서 수정합니다.

모든 원반에는 최고 레벨이 없습니다. 1~5레벨은 종류별 개별 값이고 이후에는 공통 무한 강화 테이블로 공격력·크기·속도를 계속 높입니다. 쿨타임은 안전 하한까지만 감소하지만 다른 능력과 강화 레벨은 계속 증가합니다.

## 승리, 보상, 난이도

난이도는 드롭다운 하나에서 선택합니다. easy만 처음 열려 있고 다음 순서로 진행합니다.

easy → normal → hard → harder → insane → easy demon → medium demon → hard demon → insane demon → extreme demon → easy god → medium god → hard god → insane god → extreme god

각 난이도에서 누적 20승을 해야 바로 다음 난이도가 열립니다. 19승까지는 열리지 않습니다. 열린 전투는 계속 재도전할 수 있습니다.

한 난이도 안에서도 1번째부터 20번째까지 전투 강도가 달라집니다. 승리할 때마다 다음 전투의 적 HP·공격력·이동 속도가 오르며, 일정 승수마다 적 수와 적 원반 레벨도 오릅니다. 패배하면 승리 수가 늘지 않으므로 같은 전투 단계에 다시 도전합니다. 20승 이후 재도전은 해당 난이도의 20번째 강도를 유지합니다. 증가식은 `battle-stage-rules.json`에서 관리합니다.

쿠키 보상은 난이도별 첫 승리 한 번에만 지급합니다. 같은 난이도의 2번째 이후 승리는 해금용 승리 횟수에는 포함되지만 쿠키 보상은 없습니다. 보상 수령 여부와 승리 횟수는 별도로 저장합니다.

난이도가 높을수록 테이블에 따라 적 수, HP, 공격 배율, 이동 속도와 적 원반 레벨이 오릅니다. 전 난이도의 기본 HP·공격력·이동 속도는 초기 프로토타입보다 소폭 상향한 값이며 `difficulties.json`에서 직접 조정합니다. 적은 플레이어 원반을 회피하지 않습니다.

## 몬스터 도감

전투에서 처음 만난 몬스터는 도감에서 해금되고 `NEW`를 표시합니다. 도감 화면을 확인하면 NEW 상태를 해제합니다. 아직 만나지 않은 몬스터는 이름을 `?????`로 표시합니다. 해금된 항목은 이름, 체력, 보상 쿠키, 설명을 보여줍니다.

## 설정과 피드백

사운드와 진동은 각각 켜고 끌 수 있습니다. 효과음은 1~5의 다섯 단계이며 단계 버튼을 누르면 쿠키 클릭음을 해당 볼륨으로 미리 듣습니다. 쿠키 클릭음은 Kenney UI Audio의 CC0 원본을 앱에 포함하여 오프라인 재생합니다. 전투 승리·패배 결과가 뜰 때는 종료 효과음을 재생하지 않습니다.

## 저장 항목

- 현재·누적 쿠키
- 모든 쿠키 업그레이드 단계
- 소유 원반 목록, 종류별 원반 단계, 장착 원반
- 종류별 쿠키봇 수
- 선택 난이도, 최고 해금 난이도, 난이도별 승리 수
- 난이도별 최초 보상 수령 여부
- 발견 몬스터와 NEW 상태
- 사운드 ON/OFF, 5단계 볼륨, 진동 ON/OFF

진행은 AsyncStorage에 자동 저장하며 앱을 다시 실행하면 복구합니다.
